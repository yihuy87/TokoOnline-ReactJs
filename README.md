# Web Toko Online Sederhana

sebuah Web Toko Online sederhana yang dibuat menggunakan framework React JS.

Component terdiri dari :

1. App.js
2. Navigation.js
3. Slider.js
4. ContentWeb.js

beberapa Library yang terinstal :

1. Bootstrap
2. Favicon
3. Fontawesome

Gambar dan pic diambil dari :

1. Pexels.com
2. Shopee.com
